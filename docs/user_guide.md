# Руководство пользователя

## Первый запуск

1. Запустите `YandexDiskManager.exe`
2. В меню выберите **Файл → Авторизация**
3. Нажмите **"Получить код"** — откроется страница Яндекса
4. Войдите в свой аккаунт и нажмите **"Разрешить"**
5. Скопируйте код из адресной строки (после `code=`)
6. Вставьте код в поле и нажмите **"Получить токен"**

## Основные действия

| Действие | Как сделать |
|----------|-------------|
| Просмотр файлов | Двойной клик по папке |
| Назад | Кнопка "Назад" вверху |
| Скачать файл | Правый клик → "Скачать" |
| Загрузить файл | Меню "Диск" → "Загрузить файл" |
| Создать папку | Меню "Диск" → "Создать папку" |
| Удалить файл | Правый клик → "Удалить" (только Admin) |

## Горячие клавиши

- `F5` — обновить
- `Ctrl+F` — поиск
- `Ctrl+D` — скачать выбранный файл
- `Delete` — удалить
- `Ctrl+Q` — выход

## Роли

- **Admin** — всё доступно
- **Manager** — просмотр, загрузка, теги (без удаления)
- **Viewer** — только просмотр
